《OKR：源于英特尔和谷歌的目标管理利器》读书笔记

Xu Quan（Nikkkki）
2019-02-14
读书笔记
OKR, 效率提升

最近公司要开始做新一年的OKR规划，之前没有接触过，所以抽空看了《OKR：源于英特尔和谷歌的目标管理利器》，只记录了一下对我有用的内容。以下是笔记：

OKR：源于英特尔和谷歌的目标管理利器

Description: OKR introduction book
Link: https://hbr.org/2014/06/how-to-succeed-in-business-by-bundling-and-unbundling
Tags: OKR,efficiency improvement,objective management

OKR简介

  • OKR（Objectives and Key Results）是融合了一系列框架、方法和哲学后的产物

发展历程

  • 20C，“霍桑效应”
    • 工人绩效的提升是因为他们赶紧收到了公司的关注，从而增强了工作动机
  • 德鲁克：MBO(management by objectives) “目标管理”框架
    • 应对的问题：担心现代管理者在评估绩效时，并不是评判其对公司的贡献，而是依据他们个人的专业水准的高下
    • 方法：
      • 每个管理者都需要有明确的目标，目标包括他和他的单位要达成的目标
      • 目标包括长期和短期目标
      • 包括有形（经营）和无形（组织发展、社会责任等）目标
  • 安迪·格鲁夫
    • 两个基本问题
      • 我想去哪儿（objective）？
      • 我如何调整节奏以确保我正往那儿去？—— “关键结果”(Key Results)
    • 宗旨——“聚焦”
      • 目标太多会什么都做不好
      • 甄选目标，清晰传递希望做什么，不希望做什么

    • 框架
      - 更频繁的设定OKR：季度甚至月度（不以此作为单一评判员工绩效，仅仅是员工绩效的一个输入）
      - 兼顾自上而下和自下而上
      - 强调目标挑战性的重要性
      - 即使竭尽全力，也只有一半的成功机会
      - 鼓励每个人超越现有水平

OKR是什么？

  • 一套严密的思考框架
    • 检查结果要超越数字本身
    • 深入考虑数字意味着什么
  • 持续的纪律要求
    • 时间和精力上的承诺
    • 周期性地刷新OKR
    • 确认结果达成情况
    • 持续修正
    • 结果导向
  • 旨在确保员工紧密协作
    • OKR对每一个人共享——透明性
    • 与组织对齐一致
  • 精力聚焦
    • 识别最关键的业务目标
    • 通过量化的关键结果去衡量达成情况
  • 促进组织成长
  • 可衡量的贡献

目标（Objective）

  • 核心问题：‘我们想做什么’
  • 标准
    • 简洁的
    • 定性的（没有数字）
    • 有时限要求
    • 鼓舞人心的
    • 能激发团队达成共鸣的

关键结果（Key Results）

  • 回答：‘我们如何知道自己是否达成了目标的要求’
  • 标准
    • 定量描述

好的OKR能带来哪些收益？

  • 更好的沟通
  • 敏捷——更频繁的周期
  • 聚焦
  • 公开透明
  • 自下而上
  • 前瞻性思考

OKR框架

使命

  • 核心问题： 我们存在的意义是什么？
  • 方向指导，但不可达
  • 特质
    • 简单明了——体现要专注的领域
    • 激发改变的欲望
    • 长期性
    • 易于理解和沟通

  • 有一个清晰的使命，以及与之对齐的OKR（月度、季度、年度均可），可以确保短期工作与公司长期目标保持一致

愿景

  • 未来蓝图——我们的未来将往何处去
  • 组织在未来5年、10年甚至15年后成为什么样子（尽可能地具体）
  • 典型元素包括
    • 业务活动的期望范围
    • 利益关系人（客户、员工、供应商、监管机构等）如何看待公司
    • 领导力或独特竞争力
    • 强烈的价值主张
  • 标准
    • 量化且有时间约束
    • 简洁
    • 同使命报错一致
    • 可验性（避免模糊和难以理解）
    • 可行的
    • 鼓舞人心的

战略

  • 重点和优先处理的事项

  • 四个问题：

    1. 是什么在驱动我们前进？

      • “这是一家____公司”
      • 典型的组织驱动力
        • 产品和服务
          • 使用多种渠道向不同的客户群销售
          • 聚焦某一核心产品或服务
          • i.e. 可口可乐
        • 客户和市场
          • 专注于某一特定客户群
          • i.e. 强生
        • 容量或能力
          • i.e. 酒店、航空运营
          • 能力驱动型公司（拥有特定领域的专业技能，应用到任何可能的产品或市场上）
        • 技术
          • 可用于不同产品和用户的专利技术
          • i.e. 杜邦
        • 销售和分销渠道
          • “如何做”
          • 电视网络购物
        • 原始材料
          • i.e. 石油公司

    2. 我们销售的是什么？

      • 产品 / 服务 / 两者组合

    3. 谁是我们的客户

      • 重点放在哪个客户群及地区？
      • 哪些地方不值得分散注意力？

    4. 我们如何销售？

      • 如何为客户创造价值？/ 客户为什么从你这里购买东西？

  • 四个镜像

    • 社会/文化镜像
      • 考虑战略驱动力与聚焦点的切合度
    • 人力镜像——了解团队成员的技能及人才现状
    • 技术镜像
    • 财务镜像

目标

  • 近期聚焦达成的事项

关键结果

  • 如何得知我们朝目标推进了多少？

创建有效的OKR

  • 宏伟目标的标准

    • 鼓舞人心
    • 可达到
    • 以季度为周期
    • 在团队可控范围内
    • 有商业价值
    • 定性的

  • 制定KR

    • 只写关键项，而非全部罗列
      • 聚焦在能让目标取得实际进展的KR上
    • 基于结果，而非任务
      • 找出影响Objective的KR，而非任务或行动清单
    • 使用积极正向的语言进行表述
    • 简单明了
    • 考虑所有的可能性（避免自信偏差）
    • 务必指定一个责任人

  • KR的类型

  • KR打分

    • 1.0
      • 非常有野心、几乎不可能完成的结果
      • 所有KR都英国从一开始按照这一标准来写
    • 0.7
      • 很难，但是可以达成
    • 0.3
      • 一切照旧

周例会

    • 评估进度

    • 确保团队聚焦
