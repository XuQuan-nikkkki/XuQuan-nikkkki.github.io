Week 51

Xu Quan（Nikkkki）
2019-12-23

2019年转眼就只剩不到10天了。

上周做了：

完成了
  • 做了业务环节的一个方案优化，顺带学着用axure把交互方案画出来了，还挺好玩的。
  • 终于在私有的npm上发布成功了包（经历了配置失败、上传之后服务器重启导致包丢失、重新发布之后项目部署失败等各种问题），这周还要把之前放在另一个registry的包移过来。
  • 准备在前台项目里增加log日志，方便排查问题。目前暂定的方案是 redux middleware 来收集 action 。周末看了一下 middleware ，现在能够捕获所有的 action 了。还需要等后续的存储和上传方案确定了再继续往下推进。
未完成
  • 项目迁移ts的时候，每次把一个 .js 文件改为 .tsx 或 .ts 都需要重启本地的server，因为webpack打包的时候没法找到对应的文件，本来想通过 resolve 来解决，但是没成功，准备之后再找时间尝试
