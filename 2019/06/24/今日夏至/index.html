今日夏至

发表于 2019-06-24 更新于 2019-06-27 分类于 Journal 阅读次数： Disqus：

美好的周六要用来学习。

最近电脑送修了，16 年的 macbook pro，蝶式键盘出了问题，按一下 j 会出现一排 j，也是很贱了。送修的过程异常的顺利，前年在昆明的 apple store 检查过一次，说可以免费修，但是得等 15 天，当时在昆明等着转车去大理，想想来回6 小时太折腾就没修。这次提前租了一台电脑，20 天时间，结果华贸店的人说“我们这儿时间可能长一点，大概一周吧”，当场喜出望外，留下电脑开心地忍受租来的电脑每天风扇呼呼作响。

一个厉害的 Notion 玩家

最近想把很多资料都整理到一处，考虑半天，决定慢慢把印象笔记的内容转移到 Notion 上，主要是印象笔记写东西的确不是很方便。

在网上找到一个大哥分享的如何用 Notion 做规划，写的挺好的，他把笔记、时间规划、习惯养成各种内容全都整合到 Notion 上了，还写了一个 Alfred 的 workflow，非常厉害。分享一下：

  • My Weekly Notion Setup
  • Integrating Notion with Alfred
  • Week ## Template
  • GitHub - kevinjalbert/alfred-notion: A tailored Alfred Workflow that performs tasks using notion-py within the context of a specific Notion Template.

关于 Command not found

因为在用租来的电脑，所以很多地方不太顺手。今天准备记录这些内容的时候，发现hexo new命令报错command not found: hexo。之前也出现过这类报错，每次都在网上找找解决方案，但是没有仔细看到底为什么，今天找到一篇将为什么会有这个报错的文章，写的比较简单，提供了一些基础的解决方案：

  • How to Fix “Command Not Found” Errors in Mac Command Line

近期娱乐

最近被张老师安利了《这就是街舞》，从最开始吃饭的时候看着玩，到后来有些期待周末看新一期（我是叶音粉丝），再到这周末被张老师安利了《乐队的夏天》，九连真人真是太棒了，这个夏天真是太适合电视、西瓜、啤酒和熬夜了。

不过说回来，天时真的很重要，跳街舞的一定没想到这么快就被一群摇滚老炮抢了流量，前几天朋友圈还在传 NeedAYeah 的文章，昨天晚上就大半夜一群发最近乐队改编的作品的了，当然说起来最近新一季的《中国新说唱》也在播，只是声量就更小了，搞嘻哈的应该更惨了，没想到夏天过去的如此之快。

本周福利

本周（其实也包裹上周末）收到张老师趁 618 给我订的一部P30 Pro，橙色版，之前全球断货了。因为之前的 iphone 7 摄像头尤其夜间效果的确有点让人难过，所以悄悄给我买了一部，让我改善朋友圈质量。

另外张老师还偷偷买了一个 2K的曲面屏显示器，让我写代码的时候更爽一点。怎么说呢，我可能是公司代码能力最差，但是设备用最好的了吧。

一点后记

本来上周就写完了，想要坚持写点东西，然后电脑送修，租的电脑又有各种需要设置的地方，想着就用几天，就懒得弄，所以一直没发。今天终于把该弄的都弄好了，真是要加油了啊！
