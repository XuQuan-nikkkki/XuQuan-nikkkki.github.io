《OKR工作法》读书笔记

Xu Quan（Nikkkki）
2019-08-05
读书笔记
OKR, 效率提升

又到了公司一年两度写OKR的时候，上次读的OKR的书，内容已经忘了大半，像这种情况，我都总结为书写得不够好。正好看见家里有一本《OKR工作法》，应该是张老师买的，于是拿来读了一遍，想说这本的确易读很多，且方法论陈述得很清楚，很适合我这种临时抱佛脚的人。以下是做的笔记。

  • 确定目标，确保团队聚焦到重要目标上

    • 定义
      • Objectives and Key Results
      • 目标：你想要做什么
      • 关键结果：如何确认你做到了

  • 讨论关键结果，复盘OKR实施过程中的问题

    • 右上象限：目标 + 关键结果 + 信心指数

      • 每个关键结果由一个信心指数，初始值为50%，每周更新

    • 左上象限：本周Plan

      • 3~5件重要的事情，每件都能有效推进OKR
      • 按照P1 => P2 => P3 =>P4 的顺序排列
      • 不要事无巨细，而是让大家知道需要关注哪些重要的事情

    • 右下象限：状态指标

      • 完成OKR的保障（推进OKR时也需要关注这些事）
      • 可以用颜色来标记基本情况

    • 左下象限：未来四周TODO

      • 后续计划要做的重要工作

  • 评估OKR实施成果

  • 影响目标达成的关键因素

    • 没有给目标设置优先级
    • 缺乏充分沟通，没能准确理解目标
    • 没有做好计划
    • 没有把时间花在重要的事情上
    • 轻易放弃

  • OKR基本原理

    • 设定目标
      • 目标要明确方向并且鼓舞人心
      • 要有时间期限
      • 由独立的团队来执行目标
    • 关键结果
      • 振奋人心的语言 + 需要量化

  • 6大使用场景

    • 利用OKR改进周报

      • 把团队OKR作为开始，并标注完成目标的信心指数

        • 1是永远不会成功，10是稳操胜券
        • 当信心下降到3时，把它标记为红色；超过7时，标记为绿色

      • 列出上周的优先任务，并标注完成情况

        • 没有完成的任务，可以简短解释原因

      • 列出下周的优先事项

        • 列出3个重要事项（描述全面 + 具体可执行的步骤）

      • 列出风险或阻碍

      • 备注
