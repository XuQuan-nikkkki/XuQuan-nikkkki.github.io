Chrome使用技巧

发表于 2018-06-03 更新于 2018-09-13 分类于 利器 阅读次数： Disqus：

记录一些chrome的快捷操作（来自Chrome重度依赖者的备忘）

  1. 在不退出当前页面的情况下打开新链接

    • 按住command或ctrl点击链接：在新页面打开链接
    • 按住shift点击链接：在新窗口打开链接

  2. space：向下滚动一个完整页面长度
    shift + space： 向上滚动一个完整页面的长度

  3. 重新打开最近关闭的页面（可以重复操作，依次打开之前关闭的页面）
    Mac: cmd + shift + T
    Windows: ctrl + shift + T

  4. 将所有打开选项卡保存到一个书签文件夹中
    Mac: cmd + shift + D
    Windows: ctrl + shift + D

  5. 重新启动Chrome： 在地址栏输入Chrome://restart

  6. 将Chrome变成记事本

将以上内容输入到地址栏中，就可以开始打字了，还可以保存到本地
