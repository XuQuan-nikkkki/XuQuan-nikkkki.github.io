Node.js学习笔记：fs模块实现文件读写

Xu Quan（Nikkkki）
2018-05-14
node.js

用node.js读写文件时，需要引入一个新的模块fs(file system文件系统）。文件系统的操作都有异步和同步的形式，案例使用异步方法，防止阻塞。

文件读取可以配合http模块使用。
