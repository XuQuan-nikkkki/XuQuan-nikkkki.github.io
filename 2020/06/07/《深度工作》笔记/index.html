《深度工作》笔记

发表于 2020-06-07 | 分类于 读书笔记 | | 阅读次数:
字数统计: 1,150 字 | 阅读时长 ≈ 4 分钟

《深度工作》是春节放假前读完的一本书，读完之后回武汉过年，想着回北京了就把笔记给补上，没想到再回来都是半年后了。

深度工作（Deep Work）：在无干扰的状态下专注进行职业活动，使个人的认知能力达到极限。这种努力能够创造新价值，提升技能，而且难以复制。

浮浅工作（Shallow Work）：对认知要求不高的事务性任务，往往在收到干扰的情况下开展。此类工作通常不会为世界创造太多新价值，且容易复制。

深度工作是有价值的

  1. 快速学习复杂技能要求深度工作
  2. 数字网络革命的影响力是双向的——产出自己能力范围内最好的产品需要深度

深度工作假设（The Deep Work Hypothesis）：深度工作的能力日益稀少，而几乎同时，骑在社会经济中的价值也日益提升。因此，能够培养这项技能，并将其内化为工作生活核心的人，将会取得成功。

理论

深度工作是有价值的

  • 在智能机器时代获益最大的三类人：高级技术工人、超级明星、资本所有者

  • 进入有利领域的的关键核心能力：

    1. 迅速掌握复杂工具的能力
    2. 在工作质量和速度方面都达到精英层次的能力

  • 上述两种核心能力依赖于深度工作的能力

    高质量工作产出 = 时间 x 专注度

不深度工作是少见的

  • 三大商业趋势

    1. 开放式的共享工作空间
    2. 即时通讯的兴起
    3. 各种供应商在社交媒体上出现

  • 破坏深度工作的行为难以轻松衡量——度量黑洞

  • 最小阻力原则

    最小阻力原则（The Principle of Least Resistance）：在工作环境下，若各种行为对于底线的影响没有得到明确的反馈意见，我们倾向于采用当下最简单易行的行为。

  • 忙碌代表生产能力

    忙碌代表生产能力（Busyness as Proxy for Productivity）：在工作中，对于生产能力和价值没有明确的指标时，很多知识工作者会采用工业时代代表生产能力的指标，以可视的方式完成很多事情。

深度工作是有意义的

从神学、心理学、哲学角度论证深度工作的必要性。

准则

###1. 工作要深入

  • 选定你的哲学深度

  • 禁欲主义哲学 Monastic Philosophy：摈弃或最小化浮浅职责，从而实现深度工作的最大化

  • 双峰哲学 Bimodal Philosophy：把个人事迹分成两块，将某一段明确的时间用于深度追求，余下的时间做其他所有事情

    • 深度工作可以得到极端的产出，但只有主体在这项工作中投入足够的时间，实现认知强度的最大化方可实现——因此深度工作的单位时间至少是一整天

  • 节奏哲学 Rhythmic Philosophy：轻松启动深度工作的最好方法就是将其转化成一种简单的常规习惯，其目的是创造一种工作节奏，让你不需要投入精力便可决定是否需要以及何时进入深度状态

  • 新闻记者哲学 Journalist Philosophy：在日程安排中随时可插入深度工作

原则习惯化

要想使深度工作的效果最大化，就需要养成严格内化的习惯。打磨你的深度工作的习惯：

  • 你将在何处工作，工作时间多长
  • 工作开始之后你将如何继续工作
  • 你如何支持自己的工作

###2. 拥抱无聊

  • 不要不断分心，要不断专注
  • 学习冥想
  • 记一副牌

3. 远离社交媒体

  • 在你的网络使用习惯中采用关键少数法则
    • 明确个人职业和私人生活中主要的高层次目标
    • 列出为实现每一个目标而需要的两到三个重要活动
    • 用关键活动来审查正在使用的网络根据是否有实质的积极影响
  • 戒掉社交媒体
  • 不要用网络来消遣

4. 摈弃肤浅

  • 一天的每一分钟都要做好计划
  • 定量分析每一项活动的深度
  • 向老板申请浮浅工作预算
  • 5 点半之前结束工作
