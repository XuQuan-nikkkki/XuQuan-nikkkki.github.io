2020 Week 3 & 4

发表于 2020-01-26 更新于 2023-10-31 分类于 Journal 阅读次数： Disqus：

20号回武汉，那天是周一。本来想回家了赶紧把上周的情况给补上，结果回家之后形势一天一个变，然后突然就封城了，人有些懵。等我今天反应过来看日期的时候，已经又过完了一周。

图：盘点库存时处理下来的腰封

工作

20 号下了班跑去赶火车，宣布我的春节假期开始了。虽然看起来才过了几天，但是这几天信息量太大，都有点不记得之前一周干了些什么了。（还好上上周写了工作计划）

  • 拒书优化按时上线了，有一个没测出来的 bug ，是移动端在微信调整字号后会突然出现布局问题。另一个是很久远的问题，这次仓库的操作人员直接反馈来了，iPhone 在上次照片之后会出现页面自动刷新的情况，此时拍过的照片会丢失。

  • 上线了前台的拒书返回的优化，遇到了一个 bug ，原因是并非每本书的特征都配了特征照片，导致取 url 的时候 image 是 undefined 。回头看这个 bug ，有些不应该，犯了想当然的错误。在上线之前应该把数据跑一遍，确认一下数据情况。

  • 分拣的 bug 定位到了原因，是请求超时了，操作人员理解错了页面的加载提示，传递了错误的信息。

  • 前台的 log 做完了。了解了 next 导致的 state 在 action 执行前后的变化，参考了 redux-logger 的实现方式。

  • 库存项目明确了需要做的事情，赶在放假前的最后一天写完了基础代码，测了接口。这两天可以把代码写完。

  • 赶在放假前做完了书店的盘点系统。

  • 临放假前修了一个古老的点货 bug 和一个古老的审核特征 bug

  • 线下店分类写完了90%，review 了。前后端对接口的返回数据有一些不统一。这周再根据沟通结果调整一下。

生活

  • 这两周，尤其是回来之后的这一周过的很丧。Week 3 本来看起来一切按部就班，和团队吃完了年饭，计划着回武汉过年，新闻里还在说肺炎得到控制，一切都没问题。20 号的火车回到武汉，嘱咐爸爸出门一定要戴口罩，结果到武汉之后发现绝大多数人都没有戴口罩，爸爸开玩笑说，戴口罩的都是你们外地回来的。紧接着情况开始急转直下，23 号封城了，朋友圈全是新增多少病例，以及医护人员物资如何得不到保障。接着过江隧道被封了，周边城市也封城了，直到大年初一禁止机动车出行，有种被困在这座城里恐惧无处可说的感觉。连爸爸都说，之前的措施都没有吓到他，禁止机动车出行把他吓到了，你不知道这个措施到底目的是什么。昨天去超市买东西，超市里人多到排队得1个小时，全都在抢青菜，真的有种末日的感觉。认识的熟人里有人确诊了，医院没有病床提供，朋友问我现在该怎么办，似乎的确没有任何办法，只能祈祷家里人不要出事。今天早上张老师给我打电话，说科比去世了；上微博发现非洲还在闹蝗灾；老王给我发信息，说武汉开始有人遗弃动物了……似乎没有一条好的消息，世界是要完蛋了吗？

  • 昨天收到老王的消息之后，我开始反思自己的状态。不得不说，我的心理状态现在开始出现一些问题，需要自己主动干预了。所以我告诉她，之后有坏的消息不要告诉我了，我决定开始自己屏蔽负面消息。此外，之前出现了晚上刷新闻不肯睡觉的情况，从今天开始要恢复早睡早起+时间记录。

  • 昨天掏出瑜伽垫，练了一个小时瑜伽 + 5 分钟冥想，正好这段时间在家，可以重新读一遍《简单冥想术》。

  • 看完了剩下的 3 部《人生 7 年》，和 LX 聊了一下我在看里面的人物命运时，由对他们的 judgement 反观自己，不得不说，这个系列也帮我看到了很多自己观念的局限性。这周找时间来梳理一下我看这个系列时的想法。

  • 看了《非典十年祭》，现在对敌人不了解，至少了解一下敌人的叔叔。也看了之前的应对措施，反观了最近的措施。不得不说，这个关头，我觉得自己从未离政治如此近过。

  • 看了《切尔诺贝利》。在封城的武汉看这部剧真是有种说不出的黑色幽默。豆瓣评论里有一条这么说，是“体制决定了问题的爆发，也是体制决定了义无反顾地拯救”，现在看武汉何尝不是这样，只不过我们似乎还未完全发挥体制的优势。

  • 开始读《论中国》
