我的 Alfred Workflows

Xu Quan（Nikkkki）
2024-11-07
工具箱
Alfred

最近格式化了电脑，由于之前没有备份，格式化之后只能一点一点重新把电脑调回好用的状态。这篇记录一下对我来说好用的 Workflow 们。

atop: System Monitoring and Process Management

  • https://alfred.app/workflows/chrisgrieser/atop/
  • 可以方便地查看、终止进程，替代了原来的 kill process，且响应速度明显比 kill process 快
  • 可以查看、切换蓝牙设备的链接状态，替代了原来的 Dentle
  • 还可以测网速，查看和修改 DNS 信息等等，总之非常全能

Browser Tabs

  • 可以通过快捷键切换浏览器的 tabs，支持了主流的浏览器，比如 safari、chrome、vivaldi、arc、edge
  • 我的主流浏览器还是 chrome，但会搭配 arc 在不同场景下使用
  • Browser Tabs 最方便的地方在于有时候打开了 n 个标签页，挨个去查看寻找比较麻烦，chrome tabs 命令会直接把打开的标签页都罗列出来，直接选择就可以定位到了
  • 同时搭配 alfred 的 browser bookmark 功能，可以快速定位书签页，非常方便

Color Picker

  • 原来使用的取色器是 Sip，后来用 iShot 截图之后也可以通过快捷键截图，但操作起来不算特别顺滑，只是够用
  • Color Picker 的主要功能就是取色，用 cp 命令唤起，非常快捷，取色结果支持 Hex、RGB、HSL

Currency Converter

  • 因为经常要买国外的服务，非常需要汇率换算，这个 Workflow 就是用来算汇率的
  • cur 命令唤起后，输入金额 + 币种，比如 22.9 USD，再输入想换算的币种，比如 CNY，就会自动算出结果

Epoch Converter

  • 日期与时间戳换算，debug 的时候用

Homebrew

  • 可以下载、卸载应用，比较方便的点是查看包有没有下载

Link Clean

  • 用来去除 URL 中的 search params

Lorem ipsum

  • lipsum + 数字可以生成数字字母/个词/句子/段落，在排版占位的时候很有用

Open Chrome Profile

  • 切换 chrome 的账号
  • 因为我现在有两个账号，不同的账号有不同的书签，这个 workflow 可以非常方便地切换

PDF Split

  • 输入文件名 + 页码可以拆分 PDF
  • 因为现在用到 AI 比较多，有时候需要 AI 帮忙分析文章，如果一本书内容太多，可以先拆分成章节

placeholder image

  • 输入宽高可以生成固定尺寸的占位图

Quick File Access

  • 通过快捷键查看最近的文件、Download 路径下的文件等等

Switch Appearance

  • 快速切换暗色/亮色模式

Wikipedia Suggest

  • 快速搜维基百科

Word and character count

  • 选中文本后计算文本的字数
  • 小红书笔记限制标题不能超过 20 个字，这个 Workflow 是专门用来计算标题字数的
